# スタディープラスを模倣した学習記録アプリ

既存の学習記録アプリであるStudyplusを、模倣したウェブアプリです。

* このウェブアプリを作成する理由
  前回のウェブアプリ制作では、ウェブアプリケーションの流れや使われる技術を知ることができました。しかし、AIに頼りすぎて、どうしてこの技術が使わてているのかわからないまま終わってしまうことが多かったので、意味を知るために再度同じテーマで行うことにしました。

#　開発記録
本プロジェクトの開発プロセスや、学習した技術の詳細は、以下のNotionページで公開しています。
[学習記録、試行錯誤はこちら](https://www.notion.so/3065d5e4275c8049b7c5e372db6feaa7?source=copy_link)

#　概要

* 学習不足なので、のちに更新します

# 機能

* ユーザの情報の登録/削除機能
* 学習記録のデータに対するCRUD機能（生成、読みとり、更新、削除）
* 週の合計学習時間の表示（グラフ等で一目でわかるようにする）

#　技術スタック

* HTML/CSS/Javascript(フロントエンド)
* node.js(Express.js?)バックエンド)
* MySQL(データベース)
* Git/Github(バージョン管理)
